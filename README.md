ILLA
====

An implementation of a like-Lambda Calculus
-------------------------------------------

Need Python3 and [PLY](http://www.dabeaz.com/ply/)

Usage
-----

    $ python3 testing_interpreter.py <code.lambda>


Examples
--------

    $ python3 testing_interpreter.py examples/an_if.lambda
    
    if ((5=0)) ((a 2 3) a) 6
    = if ((5=0)) ((a 2 3) a) 6
    = (5=0)
    = 6
